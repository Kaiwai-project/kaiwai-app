# KAIWAI 안심 공구 이용 가이드

KAIWAI의 공동구매(안심 공구)는 **먹튀 0% · 잠수 0%**를 목표로, 포인트 보증금과 서버 강제 정책으로 보호됩니다. 공구를 이용하기 전에 아래 정책을 꼭 확인해 주세요.

---

## 💰 포인트 정책

| 구분 | 내용 |
|------|------|
| 🛡️ 총대 보증금 | 공구방 개설 시 **300P가 보증금으로 우선 차감**. 공구를 무사히 마치고 발송 완료 시 **100% 전액 반환**. |
| 🎟️ 탑승자 수수료 | 버스 탑승 시 **300P가 안심 수수료로 차감**. |
| 👑 총대 보상 | 보증금 반환에 더해, **탑승자 수수료 총합의 50%**를 수고비로 추가 지급 (추후 제휴 브랜드 쿠폰 교환 가능). |

---

## 🚨 공구 안전 정책 (필독)

### 1. 최소 출발 하한선 — 10,000엔(¥) 강제
추가 배송비 분쟁을 **원천 차단**하기 위한 핵심 규칙입니다.

- 공구방 개설 시 **최소 출발 금액은 무료 배송 기준인 10,000엔 미만으로 설정할 수 없습니다.** (DB `CHECK (minimum_goal >= 10000)`로 강제)
- 조기 출발(주문 시작)도 **누적 금액이 10,000엔 이상**일 때만 가능합니다. 미달 상태로는 출발 버튼을 눌러도 서버가 차단합니다.
- 누적 금액 = `총대 본인 물품가 × 수량` + `전체 탑승자 (단가 × 수량)` 합산(엔 기준).

### 2. 마감 시간 만료 시 자동 무산 + 300P 자동 롤백
- 공구 개설 시 마감 시간을 **12시간 / 24시간 / 48시간** 중 선택합니다.
- **만료 시각은 서버 시간(`now()`) 기준으로 강제 산정**됩니다. 클라이언트(브라우저) 시계를 조작해도 만료 시각을 위조할 수 없습니다.
- 마감 시간이 지날 때까지 **최소 출발 금액(≥10,000엔)에 미달**하면, 공구방은 **자동으로 안전 무산** 처리됩니다.
- 자동 무산 시:
  - **탑승자 수수료 300P → 전액 자동 환불**
  - **총대 보증금 300P → 전액 자동 반환**
  - 공구방 및 장부는 안전하게 정리됩니다.

### 3. 총대 본인 옵션 수정
- **주문이 시작되기 전까지** 총대는 본인의 **물품 수량·단가 옵션**을 자유롭게 수정할 수 있습니다.
- 단, 변경은 **면세 한도(목표 금액) 범위 내**로 제한됩니다.
- 주문이 시작(마감)되면 상품 정보는 잠기며 수정할 수 없습니다.

## 👑 총대(주최자) 가이드

1. **공구방 개설(도메인 고정)** — 개설 시 입력한 상품 링크의 **사이트에서만** 탑승자가 물건을 고를 수 있도록 강제해 발주 혼선을 막습니다.
2. **정보 변경 결재** — 탑승자의 상품/수량 수정은 즉시 반영되지 않고 총대 화면에 **🚨 수정 요청 배너**로 표시되며, **승인** 후 장부에 반영됩니다.
3. **마감 및 발주** — '주문 완료' 시 방이 닫히고, 탑승자의 민감 개인정보(주소·번호)는 **총대에게만** 안전하게 표시됩니다.
4. **안전 발송** — 물건을 수령해 탑승자에게 재발송할 책임이 있으며, 완수 시 포인트 보상이 지급됩니다.

---

## 🙋‍♀️ 탑승자(참여자) 가이드

1. **버스 탑승** — 총대가 지정한 **사이트의 링크만** 입력할 수 있으며, 수령 방법에 따라 배송지 라벨이 자동으로 바뀝니다.
2. **수정은 '요청'으로** — 정보 수정 시 총대에게 **수정 요청**이 전송되고, 승인되어야 최종 반영됩니다.
3. **개인정보 보호** — 공개 장부와 별개로, 실명·주소 등 민감 정보는 발송을 위해 **총대에게만** 분리·보호되어 전달됩니다.
4. **총대 감사 보상** — 탑승 수수료의 50%는 고생하는 총대에게 보상으로 지급됩니다.
