🛡️ 총대 보증금

🎟️ 탑승자 수수료

            
버스 탑승 시 300P가 탑승 수수료로 완전히 차감됩니다.

          

          

            
👑 총대 특별 보상

            
총대님은 보증금 반환에 더해, 무거운 짐을 책임진 수고로움의 감사로 탑승자 수수료 총합의 50%를 추가 보상으로 받아요! (추후 제휴 브랜드 쿠폰 교환 가능)

          

        

      


      
      

        
🚨 공구 안전 정책 (꼭 확인!)

        

          

            
🚨 최소 출발 하한선 (10,000엔 ¥)

            
추가 배송비 분쟁을 원천 차단하기 위해, 공구방 개설 시 최소 출발 금액은 무료 배송 기준인 10,000엔 미만으로 설정할 수 없어요. 조기 출발 역시 누적 금액 10,000엔 이상일 때만 가능합니다.

          

          

            
⌛ 마감 시간 만료 시 자동 무산

            
설정한 마감 시간(12 / 24 / 48시간)이 지날 때까지 최소 출발 금액에 미달하면 공구방은 자동으로 안전 무산돼요. 이때 탑승자 수수료(300P)와 총대 보증금(300P)은 전액 자동 환불·반환됩니다.

          

          

            
✏️ 총대 본인 옵션 수정

            
주문이 시작되기 전까지 총대님은 본인의 물품 수량·단가·도수 옵션을 자유롭게 수정할 수 있어요. (단, 면세 한도 범위 내로 제한)

          

        

      


      
      

        
👑 총대(주최자) 전용 가이드

        
    
          
  1. 1
    공구방 개설(도메인 고정) — 개설 시 첫 번째 입력한 '상품주소(URL)'의 사이트에서만 탑승자가 물건을 고를 수 있게 강제 차단해 발주 혼선을 막아요.
    2. 
          
  2. 2
    정보 변경 결재 시스템 — 탑승자의 도수/수량 수정은 즉시 반영되지 않고, 총대님 화면 최상단에 🚨 [수정 요청] 배너로 떠요. 확인 후 [승인]해야 장부에 반영됩니다.
    3. 
          
  3. 3
    마감 및 발주 — '주문 완료' 시 방이 닫혀요. 탑승자의 민감한 개인정보(주소·번호)는 오직 총대님에게만 안전하게 보입니다.
    4. 
          
  4. 4
    안전 발송의 책임 — 물건을 수령해 탑승자에게 재발송할 책임이 있으며, 완수 시 포인트 보상이 주어져요.
    5. 
        

      


      
      

        
🙋‍♀️ 탑승자(참여자) 전용 가이드

        
    
          
  1. 1
    버스 탑승 — 총대님이 지정한 사이트의 링크만 넣을 수 있고, 수령 방법에 따라 배송지 입력 라벨이 자동으로 바뀌어요.
    2. 
          
  2. 2
    수정은 '요청'으로 — 정보 수정 시 총대님께 '수정 요청'이 전송되고, 승인해야 최종 반영돼요. 금액 변동 시 차액을 소통해 주세요.
    3. 
          
  3. 3
    개인정보 보호 — 장부는 투명하게 공개되지만, 진짜 개인정보(이름·주소)는 발송을 위해 총대님에게만 암호화되어 전달되니 안심하세요!
    4. 
          
  4. 4
    총대 감사 보상 — 탑승 수수료의 50%는 고생하시는 총대님께 보상으로 지급돼요. 물품을 무사히 받으셨다면 감사 인사를 잊지 마세요 💕
