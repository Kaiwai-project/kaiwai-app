# KAIWAI — Google Play 스토어 등록 메타데이터 (초안 v1)

> 실제 등록: Google Play Console → 앱 → "기본 스토어 등록정보". 아래 텍스트는 초안이며 최종 검수 후 사용하세요.
> 타깃: 1020 여성 · 일본 서브컬처 패션(양산형/지뢰계 등) 브랜드 탐색 + 안심 공동구매.

---

## 1. 텍스트 메타데이터

### 앱 이름 (최대 30자)
```
KAIWAI 카이와이 - 서브컬처 공구
```
후보:
- `KAIWAI 카이와이 - 서브컬처 공구`
- `카이와이 KAIWAI : 양산형 지뢰계`
- `KAIWAI - 일본 패션 브랜드 공구`

### 짧은 설명 (최대 80자)
```
양산형·지뢰계 브랜드 한눈에. 사기 걱정 없는 안심 공동구매까지 카이와이에서.
```
후보:
- `일본 서브컬처 패션 브랜드 탐색 + 총대 인증 안심 공구 플랫폼, 카이와이 🎀`
- `양산형·지뢰계·펀칭 좋아하는 너를 위한 브랜드 지도 & 안심 공구 🎀`

### 자세한 설명 (최대 4000자)
```
🎀 카이와이(KAIWAI) — 서브컬처 패션, 여기서 다 통해

양산형, 지뢰계, 펀칭, 로리타… 좋아하는 무드는 확실한데
어디서 사야 할지, 이 공구 믿어도 되는지 늘 고민이었나요?
카이와이는 일본 서브컬처 패션을 사랑하는 사람들을 위한
브랜드 탐색 + 안심 공동구매 플랫폼이에요.

■ 취향으로 찾는 브랜드 지도
- 양산형 · 지뢰계 등 무드별 카테고리로 브랜드를 한눈에
- 마음에 드는 브랜드는 즐겨찾기로 저장하고 빠르게 다시 보기
- 브랜드별 무드와 감성을 큐레이션한 카드로 탐색하는 재미까지

■ 사기 걱정 없는 '안심 공구'
- 총대(방장) 인증 시스템으로 믿을 수 있는 공동구매
- 방장 입금 승인 · 참여 현황을 앱에서 투명하게 확인
- 자동 합산 주문서로 뭘 얼마나 주문했는지 깔끔하게 정리

■ 놓치지 않는 이벤트 캘린더
- 좋아하는 브랜드의 팝업 · 발매 · 이벤트 소식을 한 곳에서

■ 우리끼리 통하는 피드
- 데일리룩, 하울, 취향을 공유하고 소통하는 커뮤니티

지금 카이와이에서, 취향이 같은 사람들과 더 안전하고 즐겁게
서브컬처 패션을 즐겨보세요. 🎀

문의: (지원 이메일 입력)
```

### 기타 필수 입력값
| 항목 | 값(초안) |
|------|----------|
| 앱 카테고리 | 쇼핑 (또는 라이프스타일) |
| 태그 | 패션, 쇼핑, 커뮤니티 |
| 지원 이메일 | (입력 필요) |
| 개인정보처리방침 URL | https://kaiwai.kr/privacy (실제 페이지 확인 필요) |
| 웹사이트 | https://kaiwai.kr |

## 2. 그래픽 애셋 요구사항 (직접 준비 필요 — 이미지)

| 애셋 | 규격 | 필수 | 비고 |
|------|------|:---:|------|
| 앱 아이콘 | 512×512 PNG (32bit, 알파) | ✅ | 소스 `kaiwai-app_ICON.png`(1024²) 리사이즈 |
| 피처 그래픽 | 1024×500 PNG/JPG | ✅ | 스토어 상단 배너. 캐릭터+로고 활용 |
| 스마트폰 스크린샷 | 최소 2장, 16:9 또는 9:16, 320~3840px | ✅ | 홈/브랜드/공구방/피드 화면 추천 4~8장 |
| 7인치 태블릿 | (선택) | ⬜ | 태블릿 지원 시 |
| 프로모션 영상 | YouTube URL | ⬜ | 선택 |

> 아이콘 512² 리사이즈가 필요하면 말씀해 주세요(소스 1024²로 즉시 생성 가능).

## 3. 릴리즈 체크리스트

- [ ] `android/app/build.gradle`의 `appVersionName` 확인 (현재 **1.0.0** → versionCode 10000 자동)
- [ ] 서명 keystore 백업 완료 (`kaiwai-release.keystore` + 비밀번호)
- [ ] `cd android && ./gradlew bundleRelease` → `app/build/outputs/bundle/release/app-release.aab` 업로드
- [ ] 개인정보처리방침 URL 실제 접속 확인
- [ ] 데이터 안전(Data safety) 설문 작성 (수집 정보: 계정/이메일 등)
- [ ] 콘텐츠 등급 설문 작성
- [ ] 내부 테스트 트랙으로 먼저 배포 → 검증 후 프로덕션 승격
